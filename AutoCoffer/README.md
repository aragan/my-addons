# AutoCoffer (Short README)

**Author:** Aragan  
**Add-on:** AutoCoffer.lua  

AutoCoffer automatically opens all similar containers (Coffer / Pouch / Sack / Case / Box / Parcel / Codex) from the main inventory when you manually use one of them once.

## Installation

1. Place the file:
    `AutoCoffer.lua`  
    in the path:
    `Windower4/addons/AutoCoffer/AutoCoffer.lua`

2. From within the game:
    ```
    //lua load AutoCoffer
    ```
